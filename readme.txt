This plugin help to automatically manage your LWSCache purge

== Description ==

This plugin help to automatically manage your LWSCache purge when you edit your pages, post, messages ...

It provide a way to purge all your LWSCache.

== Frequently Asked Questions ==

= Does this module work on all server? =

No, it will only work on server who use LWSCache system
